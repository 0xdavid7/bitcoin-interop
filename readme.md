# bitcoin-interop

> > This repository is the demo how to interoperate with Bitcoin.
> > It's based on the axelar-core repository.

### How to dev

1. Build the binary

```
make build
```

2. Init the genesis file

```
make init
```

3. Run the node

```
make dev
```
